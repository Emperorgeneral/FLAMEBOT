What FlameBot Does

Ingest signals

Connect Telegram and select the groups/channels you want to follow. FlameBot reads new messages and prepares them for automation.

Normalize to trade intent

Messages are parsed into structured trade intent (symbol, direction, entry, SL, TP targets) so the automation layer can behave consistently.

Execute with rules

The included MT4/MT5 EAs execute trades and apply your configured risk, symbol filters, and trade controls (close / breakeven / secure-half).

How It Works (Technical)

Signal source

Signals are typically produced by third-party providers and posted in Telegram. FlameBot does not “invent” trades — it automates execution of the messages you choose to follow.

Automation pipeline

Telegram message → desktop ingestion → backend queue/state → MetaTrader EA poll → order placement + trade-state reporting.

Failure behavior

Automation is best-effort: network latency, Telegram outages, broker execution rules, and MetaTrader configuration can affect fill quality and timing.

Platforms Supported

Telegram

MetaTrader 4

MT4 EA execution. MT4 may require WebRequest URL whitelisting in terminal settings.

MetaTrader 5

MT5 EA execution. Works with brokers/prop firms that support MT5 and allow automated trading.
